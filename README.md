## 📋 Sobre o projeto

Exercícios praticando o **paradigma lógico** com **Prolog** — disciplina de Paradigmas de Linguagem. Inclui:

- Geração de matrizes preenchidas com zeros
- Verificação de pertencimento em listas
- Encontrar maior e menor elemento de uma lista
- Operações recursivas com cláusulas e cut (`!`)

## 🛠️ Tecnologias

- **SWI-Prolog** (recomendado)

## ✅ Pré-requisitos

- [SWI-Prolog](https://www.swi-prolog.org/)

## 🚀 Como executar

```bash
swipl exercises.pl

# Exemplos de consultas:
?- geraMatriz(3, 3, M).
?- verificaSeTaNaLista(5, [1,2,3,4,5]).
?- maiorElemento([3,1,7,4,2]).
?- menorElemento([3,1,7,4,2]).
```

## 🎯 Conceitos aplicados

- Programação declarativa
- Unificação e backtracking
- Recursão sobre listas
- Operador de corte (`!`)
- Disjunção e conjunção lógica
